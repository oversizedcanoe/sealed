Sealed

Sealed is an anonymous feedback web application. It's easy:

  1. Generate a public and private code.
  2. Distribute the public code to anyone -- employees, event attendees, customers, friends.
  3. They submit completely anonymous feedback, reviews, anything.
  4. View the feedback using the original private code.
